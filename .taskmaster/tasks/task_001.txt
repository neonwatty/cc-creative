# Title: Setup Rails 8 Project with Required Dependencies
# Status: done
# Dependencies: None
# Priority: high
# Description: Initialize a new Rails 8 project with all required dependencies including Tailwind CSS, Vite via ruby_vite, and configure the database for SQLite in development and PostgreSQL in production.
# Details:
1. Install Ruby 3.3+ if not already installed
2. Install Rails 8.0+: `gem install rails -v 8.0.0`
3. Create new Rails project: `rails new claude_code_creators --css=tailwind --database=sqlite3`
4. Add ruby_vite for JS bundling: Add `gem 'ruby_vite'` to Gemfile
5. Add required gems to Gemfile:
   ```ruby
   gem 'solid_cable'
   gem 'solid_queue'
   gem 'solid_cache'
   gem 'turbo-rails'
   gem 'stimulus-rails'
   gem 'view_component'
   gem 'active_storage'
   ```
6. Run `bundle install`
7. Configure database.yml for PostgreSQL in production
8. Setup Vite configuration: `bin/rails vite:install`
9. Initialize Git repository: `git init`
10. Create initial commit with project structure

# Test Strategy:
1. Verify Rails server starts without errors
2. Confirm Tailwind CSS is properly configured by testing a simple Tailwind class
3. Verify Vite is working by creating a simple JS component
4. Test database connections for both development and production environments
5. Ensure all gems are properly installed and accessible
